# 掼蛋 AI 策略规格书（用户口径 + 公认技巧调研融合）

> 汇总用户给定策略 + 公开权威掼蛋技巧调研。各档原则一致，唯一差异是"记忆力"。
> 标注：【用户】=用户明确口径；【调研补充】=公认技巧新增；【待确认】=与用户口径有出入、需确认。
> 调研来源：知乎《掼蛋技巧100条/全攻略》、边锋《掼蛋技巧详解》、同城游、网易棋牌、CSDN 掼蛋系列等。

## 一、总目标（各档一致）【用户】
1. **自己手数最少 → 抢头游**；按记忆动态调整出牌组合。
2. **牌力差 → 帮队友头游**。
3. **压制对手多花手数**：不让对手过小牌、不让对手拿牌权。
4. 每手力求三选一保牌权：①出完仍有**回手**；②让**队友接管**；③让**对手接不动**（除非放炸）。

## 二、档位差异 = 仅"记忆力"【用户】
| 档位 | 记忆能力 |
|---|---|
| 入门 | 无记忆 |
| 中级 | 大小王 + 主牌(级牌) 的已出张数/花色（≈新手"记4王+主牌"）|
| 高级 | + 所有 10 以上的牌 |
| 大师 | 全部已出牌 |
| 宗师 | + 每张是谁出的（判断各对手强弱）|

## 三、控制牌权：顶上家、吊下家【调研补充·细化"不让对手拿牌权"】
- **顶住上家**：上家（我的上家=对手）要上手时，有大牌就压住，尽量不给他领出的机会。
- **吊住下家**：不轻易出小牌喂下家，避免下家顺势走牌。
- 总原则："能管就管不犹豫、不吝啬炸弹；不让对方连续出牌"。

## 四、领出（主动出牌）
- **单牌 / 三带二**：要出**必须有回手**，或能判断**队友能接**，否则别裸送。【用户】
  - 补：出单要"**谁打谁收**"——至少手里留一张 K 以上能把这墩收回来。【调研补充】
  - **小单张分两类**【已确认D】：①孤立弱小单（凑不成型、无回手）→ **早出腾手**（"要想上游当，必先弱单张"）；②中等、能"谁打谁收"的单 → **留作吊下家、控牌权**。
- **顺子**【已确认B】：大顺/能控的顺子**留到后边出**（越往后对手越接不动）；**小顺或无回手的散顺可以早出腾手**。
- **清三对（木板）、飞机（钢板）**：可以早期出（对手能接概率低）。【用户】
- 关键点位：**5 和 10 是顺子的关键牌**，留意它们的出没。【调研补充】

## 五、出炸
> **炸弹大小序（务必牢记）**：4 张炸 < 5 张炸 < **同花顺** < 6 张炸 < 7 张炸 < 8 张炸 < 天王炸（四王）；同级比点数。
> 即：**同花顺只能被 6 张及以上的炸或天王炸压过**；普通 4/5 张炸**压不过同花顺**。

- **必炸**：预测对手**可能 1 手（甚至 2 手）就走完** → 必炸拦截。【用户】
- 按对手剩牌张数：
  - 剩 **5 张**：若是**普通顺子/三带二**等 → 果断炸（你的炸压得过，5 张只够控两轮）。**但若判断大概率是同花顺** → 普通 4/5 张炸**压不过它**，除非你有 **6 张以上炸或天王炸**，否则**别炸**（炸不过、白费牌力）。【已确认·修正】
  - 剩 **6 / 7 张**：一般要炸（7 张多为两手牌，非炸易被走）。【用户+调研一致】
  - 剩 **4 张**：一般不必再炸。【用户+调研一致】
  - 剩 **8 / 9 张**【已确认A】：默认**偏不炸**；但按**记忆预测**——若预测对手能在 **2 手内走完**（如手握"炸+一手"）则炸拦截。
- 炸弹运用法【调研补充】：
  - **小炸**常拿不到牌权，主要用来**诱对手出更大的炸**、消耗其牌力。
  - **炸多要早炸**：用数量抢发牌/牌权优势，避免关键时炸不过别人。
  - **单独一个大炸别轻易出**，留作关键时刻保底（看对门：对手有炸则处理，没有就过）。
- 总则：是否炸 = 记忆 + 对手牌型预测，权衡**代价 vs 收益**。【用户】

## 六、强弱取舍【用户】
- 两对手一强一弱 → 可**放掉强的**（让他先走），集中对付**弱的**，省牌力（不硬压/不轻易炸强者）。

## 七、队友配合（不盖队友）
- 队友领出/控场时，**一般不跟不盖**（盖了浪费牌力、打乱队友节奏）。【用户+调研一致】
- **过小牌可以**接管：如队友出 3，过个 4/7。【用户】
  - 阈值【已确认C】：过小牌**不超过队友点数+3，且不超过 10**（如队友出 3，最多过到 6）。
- **例外**：队友的下家（对手）马上要走 → 可付**大牌/炸**拦截，防其逃掉。【用户】
- 队友**快走完 → 绝不盖**，让其走头游。【用户】
- **传牌**：自己上手时，递"队友需要、但对手不需要"的牌型（前期观察队友打出的薄弱牌型，正是其短板）。【调研补充】
- **信号牌**：队友**主动接牌**＝明确信号，表示其有把握走掉 → 全力支持、上手递牌。【调研补充】
- 救队友以**保全自己**为前提，没十足把握不拆自己成型牌救队友。【调研补充】

## 八、记牌 / 算牌 / 断张（"记忆力"的具体用法）【调研补充，支撑各档差异】
- **断张概率**（炸弹预测）：自己手上某点是**断张**（一张没有）→ 外面那 8 张**约九成会成炸**；自己某点是**单张**（外面还有 7 张）→ **约七成成炸**。据此预判对手炸弹、决定是否先发制人/留炸。
- **大牌跟踪**：大小王、主牌各打了几张，一轮单牌后即可推断剩余大概在谁手里（对应宗师"记谁出的"）。
- **残局警觉**：关键牌（如 2、A）迟迟未现 → 警惕对手手握 5 炸/6 炸。
- **残局精算**：若能推断下家就剩"一个四头炸+若干张"，可出对子等其套不住的牌型，避免喂炸。

## 九、进贡 / 还贡 / 抗贡【调研补充】
- **进贡**：非双下时尾游向头游进**最大一张**（强制）；双下时两尾游各进一张。
- **抗贡**：尾游/双下方手握**两个大王**可抗贡；抗贡后由上局头游先出。
- **还贡**：还一张 **≤10 的小牌**，不能还刚被进贡的那张。原则是**削弱对手**：
  - 避免自己因还牌多一手、避免对手多一手；
  - 避免帮对手凑成**同花顺/炸弹**、避免让其多一次上手；
  - 花色上：尽量还"自己该花色少、对手该花色多（多半无用）"的牌。

## 十、接风【调研补充】
- 某人一手走完后，若其余人都不要，则**领出权归其对家（队友）**＝接风。
- 队友接风后应利用这次"任意领出"打有利牌型（如带走队友的牌/继续控权）。

### 已确认（A–D 决议）
- **A**：8/9 张默认偏不炸，但按记忆预测"2 手内走完"则炸。
- **B**：大顺/能控的顺子留后；小顺/无回手散顺早出。
- **C**：过小牌盖队友 ≤ 队友点数+3 且 ≤10。
- **D**：孤立弱小单早出腾手；能"谁打谁收"的中等单留作吊下家。
- **炸弹序修正**：4 炸 < 5 炸 < 同花顺 < 6 炸 < … < 天王炸；同花顺只有 6 炸以上/天王炸能压。
